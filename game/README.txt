Description：
This is a simple game. We use human voice to manipulate the little flying dragon to move up and down, obtain props and avoid danger. Hope you can have fun and keep breaking records. It was made using python-pygame and basic programming concepts.

Installation instruction:
Theoretically, you can use any IDE that supports Python to run this game, such as Pycharm, VScode, Spyder, etc. the following takes Spyder as an example to show the operation process.
1. Install Anaconda from https://www.anaconda.com/products/individual
2. Install Spyder from Anaconda
3. Execute following command in terminal:
	--pip install pygame
	--pip install PyAudio
4. Run FlappyDragon.py using Spyder(from Anaconda).
